# MOMO — интернет-магазин автоакустики

Сайт [momo-eq.ru](https://momo-eq.ru): Next.js + Tailwind CSS, свой VPS, автовыкат из `main`.
Дизайн-направление — премиальный hi-end минимализм: **светлая тема по умолчанию
с переключателем на тёмную**, сигнальный оранжевый `#FF5500`, типографика
Unbounded / Manrope / JetBrains Mono, фирменный мотив — звуковая волна из логотипа.

## Панель управления

Вход — прямо на сайте: кнопка «Личный кабинет» в шапке, почта магазина и
пароль владельца. Открывается панель, а не кабинет покупателя. Запасной
адрес, если нужен, — [momo-eq.ru/admin](https://momo-eq.ru/admin). Правки
видны на сайте сразу: пересобирать и просить разработчика не нужно.

- **Заказы** — заявки с сайта: состав, контакты, статусы (новый, в работе,
  выполнен, отменён), заметка менеджера. Заказ сохраняется на сервере в момент
  оформления, даже если покупатель не дописал в WhatsApp; в шапке панели видно,
  сколько заявок ещё не разобрано.
- **Товары** — цена, остаток в штуках, категория, описание, фото. Цена и
  остаток правятся прямо в списке. Нулевой остаток закрывает покупку: на
  карточке появляется «Под заказ». Первое фото в списке становится обложкой;
  снимки обрезаются и сжимаются сами. «Скрыть» убирает товар с витрины, но
  оставляет карточку в панели — для сезонных позиций это лучше удаления.
- **Новости** — заметки журнала на главной и в разделе «Новости».
- **Сборки** — готовые комплекты: состав набирается из каталога, цена
  считается по составу и проценту скидки.
- **Контакты** — телефон, адрес, режим работы, мессенджеры, гарантия и порог
  бесплатной доставки. Реквизиты ИП правятся только через код.

Что нужно знать при работе с кодом:

- Данные читаются из папки вне репозитория (`MOMO_DATA_DIR`, на сервере
  `/home/momo/momo-data`), локально — из `data/` как раньше. Слой чтения и
  записи — `src/lib/store.ts`, доступ к данным — `src/lib/data.ts`.
- Фото отдаёт роут `/media/[...path]`: сначала загруженные через панель,
  затем исходные 145 снимков из `public/uploads`.
- Клиентским компонентам данные напрямую недоступны: цена и путь к фото —
  в `src/lib/format.ts`, контакты — через `SiteConfigProvider`.
- Перед каждой правкой панель кладёт копию файла в `backups/` (последние 20).

## Запуск (после установки Node.js)

```bash
npm install
npm run dev        # http://localhost:3000
```

## Структура

```
data/              # Исходные данные каталога (на сервере копируются в папку данных)
  products.json    #   142 товара: slug, название, бренд, категория, цена, фото
  categories.json  #   6 категорий (счётчики считаются по товарам)
  brands.json      #   MOMO, ZEUS, ANDROID, CAP, STARLIGHT
  news.json        #   новости журнала
  bundles.json     #   готовые сборки
  site.json        #   контакты, условия, реквизиты
design/prototype/  # Статичный HTML-прототип дизайна (утверждение стиля)
src/app/(shop)/    # Витрина: шапка, футер, корзина — общая обвязка ShopChrome
src/app/admin/     # Панель управления: вход, товары, новости, сборки, контакты
src/app/media/     # Отдача фото из папки данных
src/components/ui/ # Компоненты shadcn-экосистемы: tubelight-navbar, spotlight-card/GlowCard,
                   #   web-gl-shader (звуковая линия), liquid-glass-button,
                   #   SpecularButton (ogl, эффект на кнопке «Каталог»), ClickSpark (искры по клику)

src/lib/           # store (файлы данных), data (чтение), format (цены и фото для клиента),
                   #   admin-auth (пароль и сессия), image-pipeline (обработка снимков)
```

## Дизайн-токены

Сайт стартует в светлой теме; тёмная включается кнопкой в шапке и запоминается
(в прототипе — `localStorage`, в приложении — так же + cookie для SSR).

| Токен     | Светлая (по умолчанию) | Тёмная    | Роль                                  |
| --------- | ---------------------- | --------- | ------------------------------------- |
| `bg`      | `#F7F5F2`              | `#0B0B0C` | фон                                   |
| `surface` | `#FFFFFF`              | `#17171A` | карточки, панели                      |
| `line`    | `#E5E1DA`              | `#232327` | линии, рамки                          |
| `fg`      | `#141416`              | `#F5F3EF` | текст                                 |
| `muted`   | `#6E6E73`              | `#8A8A8E` | вторичный текст                       |
| `signal`  | `#FF5500`              | `#FF5500` | акцент: CTA, цены, ховеры — только он |
| `tile`    | `#FFFFFF`              | `#F2EFEA` | плитки под фото товаров               |

Фото товаров пока хостятся на старом сайте (`site.json → imageBase`);
при переходе на CMS они переезжают в Sanity CDN.
