# MOMO — интернет-магазин автоакустики

Новый сайт для [momo-eq.ru](https://momo-eq.ru): Next.js + Tailwind CSS, деплой на Vercel.
Дизайн-направление — премиальный hi-end минимализм: **светлая тема по умолчанию
с переключателем на тёмную**, сигнальный оранжевый `#FF5500`, типографика
Unbounded / Manrope / JetBrains Mono, фирменный мотив — звуковая волна из логотипа.

### Заметка про фото товаров

Фото пока грузятся напрямую со старого сайта (`momo-eq.ru/uploads/...`) через
компонент `ProductImage` с фолбэком на фирменный плейсхолдер. В песочнице
разработки нет сетевого доступа к `momo-eq.ru`, поэтому в моих превью тайлы фото
пустые/плейсхолдеры — **на вашей машине и на Vercel фото отображаются**. Баннер и
логотип лежат локально в `public/`, поэтому видны везде. В Фазе 2 фото переедут в CMS.

## Запуск (после установки Node.js)

```bash
npm install
npm run dev        # http://localhost:3000
```

## Структура

```
data/              # Seed-данные, собранные с живого momo-eq.ru
  products.json    #   145 товаров: slug, название, бренд, категория, цена, фото
  categories.json  #   6 категорий
  brands.json      #   MOMO, ZEUS, ANDROID, CAP, STARLIGHT
  news.json        #   3 новости
  site.json        #   контакты, реквизиты, база URL фото
design/prototype/  # Статичный HTML-прототип дизайна (утверждение стиля)
src/app/           # Next.js App Router: layout (шрифты), globals.css (токены)
src/components/ui/ # Компоненты shadcn-экосистемы: tubelight-navbar, spotlight-card/GlowCard,
                   #   web-gl-shader (звуковая линия), liquid-glass-button,
                   #   SpecularButton (ogl, эффект на кнопке «Каталог»), ClickSpark (искры по клику)

src/lib/           # Типы, слой данных, утилита cn() (позже данные — из Sanity)
```

## Дизайн-токены

Сайт стартует в светлой теме; тёмная включается кнопкой в шапке и запоминается
(в прототипе — `localStorage`, в приложении — так же + cookie для SSR).

| Токен     | Светлая (по умолчанию) | Тёмная    | Роль                                  |
| --------- | ---------------------- | --------- | ------------------------------------- |
| `bg`      | `#F7F5F2`              | `#0B0B0C` | фон                                   |
| `surface` | `#FFFFFF`              | `#17171A` | карточки, панели                      |
| `line`    | `#E5E1DA`              | `#232327` | линии, рамки                          |
| `fg`      | `#141416`              | `#F5F3EF` | текст                                 |
| `muted`   | `#6E6E73`              | `#8A8A8E` | вторичный текст                       |
| `signal`  | `#FF5500`              | `#FF5500` | акцент: CTA, цены, ховеры — только он |
| `tile`    | `#FFFFFF`              | `#F2EFEA` | плитки под фото товаров               |

Фото товаров пока хостятся на старом сайте (`site.json → imageBase`);
при переходе на CMS они переезжают в Sanity CDN.
